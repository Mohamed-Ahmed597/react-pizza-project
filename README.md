# 🍕 React Pizza Menu

A simple and responsive pizza menu application built with **React** and **Vite**.

## 🚀 Features

- Dynamic pizza menu
- Reusable React components
- Conditional rendering
- List rendering with `map()`
- Responsive UI
- Sold Out indicator
- Restaurant opening hours
- Clean folder structure
## 🛠️ Tech Stack

- React
- Vite
- JavaScript (ES6)
- CSS3
- JSX

## Concepts Practiced

- Functional Components
- Props
- JSX
- Conditional Rendering
- Rendering Lists
- Dynamic Class Names
- Component Composition

## ▶️ Installation

```bash
git clone https://github.com/Mohamed-Ahmed597/react-pizza-project.git

cd react-pizza-project

npm install

npm run dev
```
